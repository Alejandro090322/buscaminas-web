2. Descripción del servicio

El Sitio Web ofrece actualmente, de forma gratuita y sin necesidad de registro, un entrenador con un tutorial de reglas del buscaminas para quienes se inician en el juego.

Próximamente se incorporarán una versión jugable completa del juego, incluyendo el modo Experto, y un entrenador interactivo de patrones avanzados de resolución, con el objetivo de que el usuario aprenda a resolver partidas de forma lógica y consistente. Cuando ambas funcionalidades estén disponibles, se habilitarán puntos de conexión entre el juego y el entrenador que permitan acceder a la lección relacionada al quedar atascado en una partida real.

El servicio es y seguirá siendo gratuito. Está previsto financiarlo en el futuro mediante publicidad; mientras esta no se active, el Sitio Web no muestra ningún anuncio.
